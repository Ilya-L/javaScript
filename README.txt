API задания 2:
putArray() - принимает от вводимые данные, формирует массив и выводит его на HTML-страницу вместе с кнопкой, которая будет вызвать функцию shuffle().
shuffle() - вызывает функцию compareRandom(a, b), сортирует массив и выводит его на HTML-страницу.
compareRandom(a, b) - принимает на вход две переменные, возвращает случайное число в диапозоне от -0,5 до 0,5.

API задания 3:
solve(input) - принимает один параметр строкового типа, возвращает результат типа number.
solution() - подстваляет значение input с HTML-страницы в solve(input) и выводит возвращенное значение на HTML-страницу.
